    修改 #

    使用开发模式进行修改

    hugo server

    修改完本地会立即更改，没更改可能会有缓存

    修改完后 #

    运行hugo命令

    hugo

    会生成静态页面，生成文件在public目录下 然后进public目录

    cd public

    分别执行以下命令:

    git add .  # 添加所有更改的文件
git commit -m '备注' # 提交更改
git push # 把更改的文件上传到Github上

    最后，坐等刷新页面，有缓存可能要等一会儿才会更新
